# Bookstore SQL Database Project

## Overview
This project demonstrates a relational database design for an online bookstore using SQL.

## Features
- Relational schema with 5 tables
- Primary and foreign keys
- Sample dataset
- Analytical queries using JOIN and GROUP BY

## Database Structure
Tables:
- authors
- books
- customers
- orders
- order_items

## How to Run
1. Run schema.sql
2. Run data.sql
3. Run queries.sql

## Skills Demonstrated
SQL joins  
Database normalization  
Aggregation queries  
Relational modeling
